## 前提条件
- １部屋二人まで宿泊可能
- 連泊に制限なし
- 管理者が電話で予約を受け付け、管理者がデータを入力する
- 部屋を自由に追加でき、価格も設定できる

## 機能
- 予約状況を一覧に表示する
- 予約情報を入力する
- 予約情報を編集する
- 予約情報を削除する
- 予約が入った日を一覧に表示する
- 月、部屋毎に売上を集計する
- 部屋を追加する
